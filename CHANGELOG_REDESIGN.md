# salary-hack.com redesign report

## 実施概要
「AIが生成したテンプレートブログ」感を排除し、30代会社員が実体験で発信する副業・投資・仕事効率化メディアとして見えるよう、トップページ・共通コンポーネント・CSS・カテゴリページを再設計しました。

## 主な変更点

### 1. ヒーローエリア刷新
- 旧来の単純なヒーローバナーを廃止
- FEATURED ARTICLE構成に変更
- 左：媒体説明・CTA・コンセプト
- 右：注目記事カード
- ファーストビューで「誰が何を発信しているメディアか」が伝わる構造に変更

### 2. 信頼性ブロック追加
- 運営歴3年
- 掲載記事50本以上
- レビュー実績20件以上
- 週1〜2回更新
をトップ直下に配置し、3秒以内の信頼判断材料を追加

### 3. 絵文字アイコンの排除
- トップページ、共通JS、記事ページ内の主要絵文字を削除
- 絵文字サムネイルを廃止
- 画像未設定時も自然に見えるグラデーション型プレースホルダーへ変更

### 4. 情報設計の再構築
トップページ順序を以下に変更。
1. FEATURED
2. 信頼性指標
3. 人気記事ランキング
4. 新着記事
5. カテゴリー
6. おすすめサービス
7. 運営者プロフィール
8. フッター

### 5. カード設計の多様化
- 特集記事：大型
- 人気記事：ランキング型
- 通常記事：中型
- 補助記事：小型
- カテゴリー導線：小型グリッド
- おすすめサービス：サービスカード
同じカードが単調に並ぶ構造を廃止

### 6. サイドバー強化
- プロフィール
- 注目サービス
- 人気記事
- カテゴリー
- 最新記事
- 広告ポリシー
を追加し、実在メディア感と回遊導線を強化

### 7. フッター強化
- PROFILE
- CATEGORY
- CONTACT
- プライバシーポリシー
- 運営者情報
を追加

### 8. カラーパレット変更
- ベース：#0F172A
- アクセント：#2563EB
- 背景：#F8FAFC
- 文字：#111827
高級感・信頼感を優先した配色へ変更

### 9. SEO / semantic HTML対応
- main / section / article / aside / footer を意識した構造へ変更
- h1 / h2 の階層を整理
- articleカードを適切に使用

### 10. 既存リンク補完
- 既存ナビにあった invest.html / tools.html / money.html が未作成だったため、カテゴリページを追加
- 既存導線のリンク切れを解消

## 期待される改善
- ファーストビューで専門メディア感が出る
- 絵文字サムネイルによる仮置き感が消える
- 情報密度が上がり、実在メディアに近い構造になる
- サイドバーとランキングにより回遊性が上がる
- 運営者情報・広告ポリシーにより信頼性が上がる
