# Squad beyond風 追加調整レポート

## 実施内容

- 参考サイトのブログトップに近い「ブログ / Salary Hack Blog」導入エリアを追加
- フォントをNoto Sans JPベースに統一し、見出しもサンセリフの太字設計へ変更
- 記事一覧にカテゴリ絞り込み風チップを追加
- 絵文字・プレースホルダー中心の見た目から、無料写真素材（Unsplash）前提のカードUIへ変更
- トップページの記事カード、FEATURED、関連記事カードに画像を反映
- Finorie記事・Notta記事の本文途中に文脈に合う画像とキャプションを追加
- 記事詳細ページのアイキャッチを大型写真に変更
- Squad beyond風の白背景、丸みのあるカード、薄い境界線、明るいブルーアクセントへ調整

## 使用画像について

無料利用できるUnsplash画像をHTML内の外部URLとして設定しています。
将来的に自社画像やCanva作成画像へ差し替えやすいよう、すべて img タグで管理しています。

## 注意

Squad beyondの完全コピーではなく、構成・余白・フォント感・画像カード設計を参考にしたSalary Hack向けの調整です。
